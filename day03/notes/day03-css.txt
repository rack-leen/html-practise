一、css介绍
    
    web标准 网页制作要遵循规范
    
    html（结构） css（表现） js（行为）
    
    css 层叠样式表  作用：对网页信息显示进行控制 就是美好html标签的
    
    二、样式表
    
    什么是样式表？就是放置css代码一个环境。因为html和css是两种不同的语言，进行“沟通”，需要“桥梁”，样式表就是起了桥梁的作用
    
    样式表分为三大类：
    
    1、内部样式表     使用style标签
    
    2、外部样式表     使用link
    
    3、内联（行内）样式表  直接写在代码行内
    
    注意点：无论是style还是link都是标签，只不过它具有放置css代码的功能而已
    
    三、css语法
    
    组成：选择器（选择符）和声明组成
    
    选择器（选择符）其实是一种方法，重在选择两个字，作用：选取标签给其设置样式
    
    声明分为属性和属性值，声明是放在花括号里面的
    
    属性和属性值是用冒号连接的，用分号结束的

    四、css选择器
	
	元素选择器/类型选择器
	例子:
	div{
			height:200px ;
			width:100px ;
			color:red ;
			border:1px solid red ;
			font-size:12px ;
		}
